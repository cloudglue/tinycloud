# Tinycloud

Minimal agentic CLI for working with video files and code. Powered by [Cloudglue](https://cloudglue.dev).

## Install

### npm

```bash
npm install -g @cloudglue/tinycloud
```

### curl

```bash
curl -fsSL https://app.cloudglue.dev/tinycloud.sh | bash
```
